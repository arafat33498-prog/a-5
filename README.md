2.What is the spread operator (...)?

Ans:Spread operator JavaScript এ ব্যবহার করা হয় array বা object এর elements গুলোকে expand (ছড়িয়ে) দেওয়ার জন্য।

3.What is the difference between map(), filter(), and forEach()?

Ans:map(), filter(), এবং forEach() — তিনটিই JavaScript এর array method, কিন্তু কাজ আলাদা।

map()
প্রতিটি element এর উপর কাজ করে
নতুন array return করে

filter()
condition অনুযায়ী element select করে
নতুন array return করে

forEach()
প্রতিটি element এর উপর কাজ করে
কিছু return করে না
